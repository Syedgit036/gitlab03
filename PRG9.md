# PROGRAM 9

## Full Web Application Penetration Testing

### Use OWASP ZAP to Scan the Application and Identify the Vulnerabilities

### Solution

## Step 1: Install OWASP ZAP

Type the following command:

```bash
sudo apt install zaproxy
```

## Step 2: Scan the Application

- Once the tool is installed, open it and select:

```text
Automated Scan
```

- Select a working URL and paste it into the URL bar.

- Then press:

```text
Attack
```
```text
- Allow it to scan for a while and go to the **Alerts** section.

- Select any vulnerability and double-click on the URL.

- A dialog box will appear containing details of the attack such as:
```
```text
URL
Risk
Confidence
Description
Solution
Reference
```

## Step 3

- Copy the output generated by OWASP ZAP.

## Result

Successfully performed a web application vulnerability scan using OWASP ZAP and identified the vulnerabilities reported in the Alerts section.
